# Smart Resume Builder

A MERN Stack Resume Builder application that helps users create, manage, and download resumes.

## Features

- Create Resume
- View Resume Dashboard
- Edit Resume
- Delete Resume
- Download Resume as PDF
- ATS Score Checker

## Tech Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Installation

### Clone Repository

```bash
git clone https://github.com/varsha-24Yerraguntla/smart-resume-builder.git
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd server
npm install
npm start
```

## Environment Variables

Create a `.env` file inside the server folder:

```env
MONGO_URI=mongodb+srv://varsha:varsha123@cluster0.mbuxdsq.mongodb.net/?appName=Cluster0
PORT=5000
```
